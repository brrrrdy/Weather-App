![Weather App screenshot](https://tomalvarez.xyz/assets/scrn_weather-CtXMEvM5.webp)

## REQUIREMENTS

- Search for and display weather for a specific location
- Toggle between Fahrenheit and Celsius
- UI that changes dynamically based on weather conditions
- Built with async/await
- Process and filter API JSON data into only the fields the app needs

## ABOUT

A weather forecast app built as part of The Odin Project curriculum, designed to practise working with external APIs, asynchronous JavaScript, and dynamic UI updates.

The app fetches live weather data from the Visual Crossing API based on a user-supplied location, processes the JSON response to extract only the relevant fields, and displays the results with a UI that adapts visually to current conditions. Temperature can be toggled between Fahrenheit and Celsius.

API key security was not a consideration for this project, and is thus exposed client-side.

Built with Webpack, using dynamic imports to handle weather icons efficiently.
